# Chat 기능

## Chat이란?

GitHub Copilot Chat은 대화형 인터페이스로 코드 작성, 설명, 디버깅을 도와줍니다.

## 주요 기능

### 1. 코드 설명
- 선택한 코드의 의미 설명
- 복잡한 로직 이해하기

### 2. 코드 생성
- 자연어로 요구사항 설명
- Copilot이 코드 생성

### 3. 버그 찾기
- 코드의 문제점 식별
- 수정 방법 제안

## 사용 방법

1. VS Code에서 `Ctrl+Shift+I` (또는 `Cmd+Shift+I`)
2. Chat 패널에 질문 입력
3. 제안받은 코드 검토 및 적용

## 실습

Chat을 사용하여 다음을 시도해보세요:
- "배열을 정렬하는 함수 만들어줘"
- "이 코드 설명해줘" (코드 선택 후)
- "이 함수의 테스트 코드 작성해줘"
