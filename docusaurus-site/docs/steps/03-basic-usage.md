# 기본 사용법

## 코드 자동완성

GitHub Copilot은 코드를 작성하는 동안 실시간으로 제안을 제공합니다.

### 사용 방법

1. 코드를 입력하기 시작하면 자동으로 제안이 나타납니다
2. `Tab` 키를 눌러 제안을 수락합니다
3. `Esc` 키를 눌러 제안을 거부합니다

## 주석으로 코드 생성하기

주석을 작성하면 Copilot이 해당 기능을 구현하는 코드를 제안합니다.

### 예제

```javascript title="기본 함수 생성 예시"
// 두 숫자를 더하는 함수
function add(a, b) {
    return a + b;
}
```
