# 💻 기본 사용법 

## 코드 자동완성

GitHub Copilot은 코드를 작성하는 동안 실시간으로 제안을 제공합니다.

### 사용 방법

1. 코드를 입력하기 시작하면 자동으로 제안이 나타납니다
2. `Tab` 키를 눌러 제안을 **수락**합니다
3. `Esc` 키를 눌러 제안을 **거부**합니다

## 주석으로 코드 생성하기

주석을 작성하면 Copilot이 해당 기능을 구현하는 코드를 제안합니다.

### 예제

example.py 파일을 생성하고 vscode 로 열어주세요. 다음과 같이 주석만 작성하고 엔터키를 누르면 코드 제안이 됩니다. `Tab` 키를 눌러 제안을 수락하세요.:
```python title="기본 함수 생성 예시"
# 두 숫자를 더하는 함수
def add(a, b):
    return a + b
```
