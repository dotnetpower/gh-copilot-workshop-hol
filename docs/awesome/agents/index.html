Awesome Agents

전문화된 GitHub Copilot 에이전트는 MCP 서버와 통합되어 특정 워크플로우와 도구에 대한 향상된 기능을 제공합니다. 특정 목적에 맞게 지침을 사전 설정하여 사용자 프롬프트를 최소화 할 수 있습니다.

개요​

커스텀 에이전트는 특정 작업이나 도메인에 특화된 AI 어시스턴트입니다. 이들은 MCP(Model Context Protocol) 서버와 통합되어 GitHub Copilot의 기능을 확장하고 전문화된 지원을 제공합니다.

사용 방법​

Copilot 코딩 에이전트 (CCA)​

  1. GitHub 이슈를 Copilot에 할당할 때
  2. 제공된 목록에서 커스텀 에이전트 선택
  3. 에이전트가 해당 작업에 대한 전문화된 지원 제공

VS Code​

  1. 에이전트 세션에서 커스텀 에이전트 활성화
  2. Plan, Agent 등 기본 제공 에이전트와 함께 사용
  3. 특정 워크플로우에 맞춤화된 지원 받기

Copilot CLI​

곧 지원 예정입니다.

주요 기능​

  • MCP 서버 통합: 외부 도구 및 서비스와의 원활한 연결
  • 전문화된 워크플로우: 특정 작업에 최적화된 지원
  • 컨텍스트 인식: 프로젝트 컨텍스트를 이해하고 관련 제안 제공
  • 확장 가능: 필요에 따라 새로운 에이전트 추가 가능

사용 가능한 에이전트​

Awesome GitHub Copilot 리포지토리에는 다양한 도메인과 사용 사례를 위한 커스텀 에이전트가 포함되어 있습니다:

  • DevOps 자동화
  • 보안 분석
  • 데이터베이스 관리
  • 클라우드 인프라
  • 관찰성 및 모니터링
  • 기능 플래그 관리
  • CI/CD 파이프라인
  • 마이그레이션 도구
  • 성능 최적화
